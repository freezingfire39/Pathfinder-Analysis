1) Download and install Python from this link: https://www.python.org/downloads/
   NOTE: Make sure to check Add Python to PATH while installing.

2) Install python libraries listed in requirements.txt file
   Use this command in Command Prompt: pip install library_name

3) Download chromedriver according to your OS and Chrome Version(Check in Settings > About) from 
   this link: https://chromedriver.chromium.org/downloads

4) Unzip chromedriver file. Copy and Paste chromedriver file in the same folder as the script/code/bot.
